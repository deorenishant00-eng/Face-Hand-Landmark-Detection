# Face-Hand-Landmark-Detection
A real-time computer vision project that detects and tracks face and hand landmarks using MediaPipe Holistic and OpenCV. The application captures webcam input, identifies facial mesh and hand keypoints, and displays them with live FPS for efficient real-time performance.
# MediaPipe Face and Hand Landmark Detection

A real-time computer vision project built with Python, MediaPipe, and OpenCV that detects facial landmarks and hand landmarks using a webcam. The application processes live video, identifies face mesh and hand keypoints, and displays them with an FPS counter.

---

## Features

- Real-time webcam capture
- Face Mesh Detection
- Left Hand Landmark Detection
- Right Hand Landmark Detection
- Live FPS Counter
- Fast and lightweight implementation
- Built using MediaPipe Holistic

---

## Technologies Used

- Python 3.x
- OpenCV
- MediaPipe
- NumPy

## Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/mediapipe-face-hand-landmark-detection.git
```

### Navigate to the project

```bash
cd mediapipe-face-hand-landmark-detection
```

### Create Virtual Environment

Windows

```bash
python -m venv myenv
```

Activate

```bash
myenv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Project

```bash
python app.py
```
